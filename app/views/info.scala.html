Hi, this is a CRUD movie list app. Use the following commands:

  • GET /allMovies - Getting JSON with all movies from the database
  • GET /someMovie?id=* - Getting a JSON movie by its id
  • PUT /movie/change?id=* - Changing the name of a movie by its id
  • DELETE /movie/delete?id=* - Deleting a movie by its id
  • POST /movies - Adding a movie to the database
* its id movie
